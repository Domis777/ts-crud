# __Fullstack CRUD application__

## _Required Programs_
  * Node v18.x.y or higher

## _Instalation_
Open console or terminal and run:
```
  npm init
```
or
```
  npm i
```
## _Convert files_
Open console or terminal and run:
```
  npm run build
```
## _Server launch_
Open console or terminal and run:
```
  npm run start
```
